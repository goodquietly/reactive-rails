# Маленькая API позволяет создавать и хранить контакты(Имя и email)

Знакомсто с популярной `JS` библиотекой `React`.

### Используемые технологии.

```

- Ruby 3.0.1;

- Rails 6.1.6;

- БД Postgres;

- react;

- react-router;

- react-i18next;

- react-bootstrap;

- axios;

- hooks;

- postman.

```
